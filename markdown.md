# Markdown
## 마크다운의 장단점
1. 마크다운의 장점
    - 문법이 쉽다.
    - 관리가 쉽다.
    - 지원 가능한 플랫폼과 프로그램이 다양하다.
2. 마크다운의 단점
    - 표준이 없어 사용자마다 문법이 상이할 수 있다.
    - 모든 HTML 마크업을 대신하지 못한다.
## 마크다운 문법
1. 제목(Header)
    - # 제목1
    (# 제목1)
    - ## 제목2
    (## 제목2)
    - ### 제목3
    (### 제목3)
    - #### 제목4
    (#### 제목4)
    - ##### 제목5
    (##### 제목5)
    - ###### 제목6
    (###### 제목6)
